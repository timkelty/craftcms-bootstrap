Class fusionary\craftcms\bootstrap\helpers\Config

All Classes | Properties | Methods | Constants
Inheritancefusionary\craftcms\bootstrap\helpers\Config

Bootstrap Config Helper

Use this class in your config files to dynamically map environment variables to Craft settings.

Example: config/general.php

// export CRAFT_ALLOW_AUTO_UPDATES=true;
use fusionary\craftcms\bootstrap\helpers\Config;

return Config::mapMultiEnvConfig([
    '*' => [
        'allowAutoUpdates' => true,
    ],
    'production' => [
        'allowAutoUpdates' => false,
    ]
]); // â†’ ['*' => ['allowAutoUpdates' => true], 'production' => ['allowAutoUpdates' => true]]

getEnv() public static method

Get the value of an environment variable based on a config setting.

Values are converted to converted to their respective data types using Cekurte\Environment.

use fusionary\craftcms\bootstrap\helpers\Config;

// export CRAFT_DEV_MODE=true
Config::getEnv('devMode', false); // â†’ (bool) true

// export CRAFT_MY_SETTING=someSetting
Config::getEnv('mySetting'); // â†’ (string) 'someSetting'

// export PLUGIN_MY_SETTING=100
Config::getEnv('mySetting', null, 'PLUGIN_'); // â†’ (int) 100

See also https://github.com/jpcercal/environment#examples.
